This is a simple calculator built using Python. It allows users to perform basic arithmetic operations through the terminal.



Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Handles invalid input
- Prevents division by zero



Tech Used
- Python



How to Run

1. Make sure Python is installed
2. Open terminal in the project folder
3. Run:

python my-first-py-project.py
